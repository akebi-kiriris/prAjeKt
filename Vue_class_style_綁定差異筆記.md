# Vue :class 與 :style 綁定差異筆記

## 1. :class 綁定
- 可以接受字串、陣列或物件。
- 給空字串 `''` 代表沒有 class，這是合法且常見的寫法。
- 例如：
  ```vue
  <div :class="isLogin ? 'active' : ''"></div>
  ```

## 2. :style 綁定
- 建議給「物件」或「物件陣列」。
- 給空物件 `{}` 代表沒有任何行內樣式，這是官方推薦的寫法。
- 例如：
  ```vue
  <div :style="isLogin ? { color: 'red' } : {}"></div>
  ```
- 若給空字串 `''`，雖然瀏覽器能接受，但型別不一致，且不利於後續合併多個 style。

## 3. 實務差異
- `:class` 綁定空字串沒問題，因為 class 本來就是字串。
- `:style` 綁定建議用空物件 `{}`，因為 style 在 Vue 裡預期是物件格式。

## 4. 小結
- class 綁定空字串 = 沒有 class
- style 綁定空物件 = 沒有行內樣式
- 這樣寫法最安全、最符合 Vue 官方建議。
